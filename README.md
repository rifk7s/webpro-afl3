## 🚀 Getting Started

### 📋 Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v16.0 or higher) - [Download here](https://nodejs.org/)
- **npm** (comes with Node.js) or **yarn** as package manager
- **Git** for version control - [Download here](https://git-scm.com/)
- A modern web browser (Chrome, Firefox, Safari, Edge)

### ⚡ Quick Start

1. **📥 Clone the repository:**
   ```bash
   git clone https://github.com/rifk7s/webpro-afl3.git
   cd webpro-afl3
   ```

2. **📦 Install dependencies:**
   ```bash
   npm install
   ```

3. **🎨 Build & watch Tailwind CSS:**
   ```bash
   npm run test
   ```
   
   This command runs:
   ```bash
   npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
   ```

4. **🌐 Open in browser:**
   - Open `index.html` in your preferred browser
   - Or use a local server like Live Server for better development experience

### 🛠️ Development Workflow

```bash
# Start Tailwind CSS compilation in watch mode
npm run test

# The build process will:
# ✅ Process src/input.css
# ✅ Generate src/output.css  
# ✅ Watch for changes automatically
# ✅ Apply Tailwind utilities and custom styles
```

### 📂 File Structure for Development

```
📝 Edit these files:
├── 🏠 index.html, about.html, contact.html, projects.html
├── ⚙️ script.js (JavaScript functionality)  
├── 🎨 src/input.css (Tailwind CSS input)
└── 🖼️ img/ (add your images here)

🔄 Auto-generated files:
├── 🎨 src/output.css (compiled Tailwind CSS)
└── 📁 node_modules/ (dependencies)
```

### 🔧 Customization

- **🎨 Styling**: Edit `src/input.css` and `tailwind.config.js`
- **⚙️ Functionality**: Modify `script.js` for interactive features
- **📝 Content**: Update HTML files with your information
- **🖼️ Assets**: Replace images in `img/` directory with your own
